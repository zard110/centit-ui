通过JS生成树

第二棵树只需要一个空的ul元素，通过设置url从后台请求数据生成树

    属性 作用
    id 节点id
    text 节点标题
    iconCls 节点图标的class名
    state 节点状态：closed-关闭 open-展开
    children 节点的子元素(数组)
